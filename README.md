# Lunexa Website

This repository contains a simple multi-page website for the Lunexa AI crypto trading application. The project showcases a luxury design with animated sections, glassmorphic cards and a responsive navigation drawer.

Pages included:

- `index.html` – main landing page with feature sections
- `about.html` – background information on Lunexa
- `contact.html` – form to reach the team
- `blog.html` – sample blog with four articles

Styles and interactions are handled by `styles.css` and `script.js`.
